📄 HTML · 🎨 CSS · ⚡ JavaScript

Три столпа фронтенда: их уникальные функции и возможности

HTML

Основа — структура документа.

  • Создание семантической сетки (<header>, <article>, <section>)
  • Формы и поля ввода (интерактив без JS)
  • Встраивание мультимедиа <video>, <audio>, <canvas>
  • Гиперссылки и навигация по сайту
  • Атрибуты для доступности (ARIA, alt)
  • Загрузка ресурсов: картинки, шрифты, стили
<!-- Пример структуры -->
<article class="card">
  <h3>Заголовок</h3>
  <p>Текст абзаца</p>
</article>

CSS

Визуал — внешний вид и анимация.

  • Точное позиционирование (Flexbox, Grid, absolute)
  • Адаптивный дизайн (media queries)
  • Анимации и переходы (@keyframes, transition)
  • Трансформации (scale, rotate, translate)
  • Тени, градиенты, фильтры, переменные (custom properties)
  • Псевдоклассы (:hover, :focus) и псевдоэлементы (::before)
.button {
  background: linear-gradient(45deg, gold, orange);
  transition: transform 0.2s;
}
.button:hover { transform: scale(1.05); }

JavaScript

Функциональность — интерактив и логика.

  • Манипуляция DOM: создание, удаление элементов
  • Обработка событий (клик, ввод, загрузка)
  • Асинхронность: fetch, async/await, таймеры
  • Хранение данных: localStorage, cookies
  • Работа с массивами, объектами, функциями высшего порядка
  • Валидация форм и динамическая подгрузка контента
// изменить текст элемента
const title = document.querySelector('h1');
title.textContent = 'Привет, мир!';
// слушатель клика
button.addEventListener('click', () => { ... });
